### Core Components

#### 1. **Data Pipeline (ETL)**
- **Extract**: Multi-source blockchain data collection from 4+ APIs
- **Transform**: Data normalization and enrichment
- **Load**: Structured data storage and session management

#### 2. **Data Sources Integration**
- **Moralis API**: Primary blockchain data provider
- **Etherscan API**: Transaction verification and method signatures
- **Alchemy API**: Enhanced blockchain data and analytics
- **Infura API**: Backup blockchain data source
- **CoinGecko API**: Token metadata and pricing

#### 3. **Data Enrichment Layer**
- **Address Intelligence**: ENS domains, Unstoppable Domains, net worth analysis
- **Transaction Context**: Method signature decoding, function descriptions
- **External Intelligence**: Arkham Intel integration, Metasleuth labels

#### 4. **Analytics Engine**
- **Price Impact Calculator**: Multi-timeframe price impact analysis (1h, 2h, 4h, 8h, 12h)
- **OHLCV Data Processing**: Real-time price data with transaction markers
- **Statistical Analysis**: Return calculations and trend analysis

#### 5. **AI Intelligence Module**
- **Gemini LLM Integration**: Natural language transaction summaries
- **Context-Aware Analysis**: Combines transaction data with price impact
- **Automated Reporting**: Generate human-readable insights

#### 6. **Visualization & UI**
- **Interactive Charts**: Plotly-powered candlestick charts with transaction markers
- **Real-time Dashboards**: Streamlit-based responsive interface
- **Data Tables**: Comprehensive transaction and analysis data display


### Technical Stack

#### Backend Technologies
- **Python 3.13**: Core application language
- **Streamlit**: Web application framework
- **Pandas**: Data manipulation and analysis
- **Plotly**: Interactive data visualization
- **LangChain**: AI/LLM integration framework

#### Data & APIs
- **Moralis API**: Primary blockchain data
- **Etherscan API**: Transaction verification
- **Alchemy API**: Enhanced blockchain analytics
- **Infura API**: Backup blockchain data
- **CoinGecko API**: Token metadata
- **Gemini API**: AI/LLM capabilities

#### External Intelligence
- **Arkham Intel**: Wallet labeling and exchange identification
- **Metasleuth**: Address intelligence and risk assessment
